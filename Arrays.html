Arrays in Java

What is Array?

In Java, an array is a data structure that stores a fixed-size sequential collection of elements of the same type.

Benefits of Using Arrays

  • Allows storing multiple values of the same type in a single variable.
  • Provides random access to elements using index.
  • Facilitates efficient data manipulation and processing.

Drawbacks of Using Arrays

  • Fixed size: Arrays have a fixed length that cannot be changed once declared.
  • Homogeneous elements: Arrays can only store elements of the same data type.
  • No built-in methods for dynamic operations like resizing.

Ways to Declare Arrays

In Java, arrays can be declared using the following methods:


// Method 1: Declaration with size
int[] arr1 = new int[5];

// Method 2: Declaration with initialization
int[] arr2 = {1, 2, 3, 4, 5};

// Method 3: Declaration using 'new' keyword and initializing later
int[] arr3;
arr3 = new int[]{10, 20, 30};

Types of Arrays

Arrays in Java can be categorized into the following types:

  • Single-Dimensional Arrays: Arrays with a single row or column.
  • Multi-Dimensional Arrays: Arrays with multiple rows and columns, like 2D or 3D arrays.
  • Variable Length Arrays (Introduced in Java 5): Arrays whose length can be determined at runtime.
  • Arrays of Objects: Arrays that store references to objects.
  • Arrays of Arrays (Multi-Dimensional): Arrays whose elements are themselves arrays.

Array Manipulation Methods

Here are some common methods used to manipulate arrays:

  • Sorting: Use `Arrays.sort()` method. 
    
int[] numbers = {5, 1, 4, 2, 3};
Arrays.sort(numbers);
System.out.println(Arrays.toString(numbers)); // Outputs: [1, 2, 3, 4, 5]
  • Searching: Use `Arrays.binarySearch()` method. 
    
int[] numbers = {1, 2, 3, 4, 5};
int index = Arrays.binarySearch(numbers, 3);
System.out.println("Index of 3: " + index); // Outputs: Index of 3: 2
  • Copying: Use `System.arraycopy()` method. 
    
int[] source = {1, 2, 3, 4, 5};
int[] destination = new int[source.length];
System.arraycopy(source, 0, destination, 0, source.length);
System.out.println(Arrays.toString(destination)); // Outputs: [1, 2, 3, 4, 5]
  • Comparing: Use `Arrays.equals()` method. 
    
int[] arr1 = {1, 2, 3};
int[] arr2 = {1, 2, 3};
boolean isEqual = Arrays.equals(arr1, arr2);
System.out.println("Arrays are equal: " + isEqual); // Outputs: Arrays are equal: true
  • Filling: Use `Arrays.fill()` method to fill array elements with a specific value. 
    
int[] arr = new int[5];
Arrays.fill(arr, 10);
System.out.println(Arrays.toString(arr)); // Outputs: [10, 10, 10, 10, 10]
  • Converting Array to List: Use `Arrays.asList()` method to convert an array to a List. 
    
String[] words = {"apple", "banana", "cherry"};
List wordList = Arrays.asList(words);
System.out.println(wordList); // Outputs: [apple, banana, cherry]
