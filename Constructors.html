Constructors in Java

What is Constructor?

In Java, a constructor is a special method that is automatically called when an object of a class is instantiated. It initializes the object and allocates memory for it.

Types of Constructors

  • Default Constructor: Constructor with no parameters provided by the compiler if no other constructor is defined.
  • Parameterized Constructor: Constructor with parameters to initialize the object with specific values.
  • Copy Constructor: Constructor that initializes an object by copying values from another object of the same class.
  • Constructor Overloading: Having multiple constructors within the same class with different parameters.

Ways to Define Constructors

Constructors in Java can be defined in the following ways:


// Method 1: Default Constructor
public class MyClass {
    public MyClass() {
        // Constructor code
    }
}

// Method 2: Parameterized Constructor
public class MyClass {
    private int value;

    public MyClass(int v) {
        this.value = v;
    }
}

// Method 3: Copy Constructor
public class MyClass {
    private int value;

    public MyClass(MyClass original) {
        this.value = original.value;
    }
}

Constructor Examples

Here are some examples of constructors in Java:


// Example 1: Default Constructor
public class Car {
    private String model;
    private String color;

    // Default Constructor
    public Car() {
        this.model = "Unknown";
        this.color = "Unspecified";
    }
}

// Example 2: Parameterized Constructor
public class Employee {
    private String name;
    private int age;

    // Parameterized Constructor
    public Employee(String n, int a) {
        this.name = n;
        this.age = a;
    }
}

// Example 3: Copy Constructor
public class Book {
    private String title;
    private String author;

    // Copy Constructor
    public Book(Book original) {
        this.title = original.title;
        this.author = original.author;
    }
}
