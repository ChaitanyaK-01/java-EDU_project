Object-Oriented Programming (OOP) with Java

Encapsulation

Explanation of encapsulation in OOP, its purpose, and benefits. Types of encapsulation: access modifiers (private, protected, public). Ways to achieve encapsulation: using accessors (getters) and mutators (setters). Example code demonstrating encapsulation in Java.


// Example demonstrating encapsulation
public class Employee {
    private String name;
    private double salary;

    public Employee(String empName, double empSalary) {
        this.name = empName;
        this.salary = empSalary;
    }

    // Getter for name
    public String getName() {
        return name;
    }

    // Setter for salary
    public void setSalary(double salary) {
        if (salary > 0) {
            this.salary = salary;
        }
    }
}

Inheritance

Explanation of inheritance in OOP, its purpose, and benefits. Types of inheritance: single inheritance, multiple inheritance (using interfaces), hierarchical inheritance, multilevel inheritance. Ways to achieve inheritance: using extends keyword for classes, implementing interfaces. Example code demonstrating inheritance in Java.


// Example demonstrating inheritance
// Parent class
class Animal {
    void sound() {
        System.out.println("Animal makes a sound");
    }
}

// Child class inheriting from Animal
class Dog extends Animal {
    void sound() {
        System.out.println("Dog barks");
    }
}

Polymorphism

Explanation of polymorphism in OOP, its purpose, and benefits. Types of polymorphism: compile-time polymorphism (method overloading), runtime polymorphism (method overriding). Ways to achieve polymorphism: method overloading, method overriding, interfaces. Example code demonstrating polymorphism in Java.


// Example demonstrating polymorphism
// Method Overloading
class Calculator {
    int add(int a, int b) {
        return a + b;
    }

    double add(double a, double b) {
        return a + b;
    }
}

// Method Overriding
class Animal {
    void sound() {
        System.out.println("Animal makes a sound");
    }
}

class Dog extends Animal {
    void sound() {
        System.out.println("Dog barks");
    }
}

Abstraction

Explanation of abstraction in OOP, its purpose, and benefits. Types of abstraction: abstract classes, interfaces. Ways to achieve abstraction: using abstract keyword, implementing interfaces. Example code demonstrating abstraction in Java.


// Example demonstrating abstraction
// Abstract Class
abstract class Shape {
    abstract void draw(); // Abstract method
}

// Implementation
class Circle extends Shape {
    void draw() {
        System.out.println("Drawing Circle");
    }
}

// Interface
interface Drawable {
    void draw();
}

// Implementation
class Rectangle implements Drawable {
    public void draw() {
        System.out.println("Drawing Rectangle");
    }
}
