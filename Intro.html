Introduction to Java

Class and Object

In Java, a class is a blueprint or template for creating objects. An object is an instance of a class. Classes encapsulate data for the object and define methods to operate on that data.

Basic Structure of a Java Program


public class HelloWorld {
    public static void main(String[] args) {
        // Your Java code here
    }
}

A Java program starts with a class definition. The `main` method is the entry point where execution begins.

Variables

Variables in Java are containers for storing data values. They have a data type and a name. Java supports different types of variables, such as:

  • Local variables
  • Instance variables (non-static fields)
  • Class variables (static fields)

Data Types

Data types specify the type of data that variables can hold. Java supports various primitive data types:

  • byte
  • short
  • int
  • long
  • float
  • double
  • boolean
  • char

Java also supports non-primitive data types, which are not defined by the programming language but are instead created by the programmer. They include:

  • Classes
  • Interfaces
  • Arrays
  • Strings
  • Wrapper classes (e.g., Integer, Double)
  • Enum types

Literals

Literals in Java represent fixed values that can be assigned to variables. Examples include numeric literals (e.g., `123`, `3.14`), character literals (e.g., `'A'`, `'B'`), and string literals (e.g., `"Hello"`, `"Java"`).

Wrapper Classes

Wrapper classes in Java are used to convert primitive data types into objects. They provide a way to work with primitive data types as objects. Benefits include:

  • Allowing null values
  • Enabling generic programming with collections (e.g., ArrayList)
  • Providing utility methods (e.g., parsing)

Operators

Operators in Java are symbols that perform operations on variables and values. They include:

  • Arithmetic operators (e.g., `+`, `-`, `*`, `/`, `%`)
  • Comparison operators (e.g., `==`, `!=`, `>`, `<`, `>=`, `<=`)
  • Logical operators (e.g., `&&`, `||`, `!`)
  • Bitwise operators (e.g., `&`, `|`, `^`, `~`, `<<`, `>>`, `>>>`)
  • Assignment operators (e.g., `=`, `+=`, `-=`, `*=`, `/=`)
