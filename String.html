Strings in Java

What is String?

In Java, a String is a sequence of characters. Strings are used to store and manipulate text-based data.

Ways to Declare String

In Java, there are two main ways to declare a String:


// Method 1: Using String literal
String str1 = "Hello";

// Method 2: Using String object
String str2 = new String("World");

Why String is Immutable?

In Java, String objects are immutable, meaning once a String object is created, its content cannot be changed. This ensures thread safety, caching, and security.

How to Make String Mutable?

To work with mutable strings in Java, you can use the `StringBuilder` or `StringBuffer` classes, which provide mutable sequences of characters.


// Example using StringBuilder
StringBuilder mutableStr = new StringBuilder("Java");
mutableStr.append(" Programming");
System.out.println(mutableStr.toString()); // Outputs: Java Programming

Methods for String Manipulation

Here are some common methods used to perform actions on strings:

  • Reversing String: Use `StringBuilder.reverse()` method. 
    
String str = "Hello";
StringBuilder reversed = new StringBuilder(str).reverse();
System.out.println(reversed.toString()); // Outputs: olleH
  • Finding Last Element of String: Use `charAt(length() - 1)` method. 
    
String str = "Java";
char lastChar = str.charAt(str.length() - 1);
System.out.println("Last character: " + lastChar); // Outputs: Last character: a
  • Finding First Element of String: Use `charAt(0)` method. 
    
String str = "Java";
char firstChar = str.charAt(0);
System.out.println("First character: " + firstChar); // Outputs: First character: J
  • Length of String: Use `length()` method. 
    
String str = "Java";
int length = str.length();
System.out.println("Length of the string: " + length); // Outputs: Length of the string: 4
  • Comparing Length of Two Strings: Use `length()` method and compare. 
    
String str1 = "Java";
String str2 = "Python";
if (str1.length() == str2.length()) {
    System.out.println("Strings have the same length.");
} else {
    System.out.println("Strings have different lengths.");
}
  • Comparing Content of Two Strings: Use `equals()` or `equalsIgnoreCase()` method. 
    
String str1 = "Java";
String str2 = "java";
if (str1.equalsIgnoreCase(str2)) {
    System.out.println("Strings are equal ignoring case.");
} else {
    System.out.println("Strings are not equal ignoring case.");
}
