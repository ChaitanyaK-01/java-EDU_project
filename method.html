Methods in Java

Method vs Function

In Java, a method is a collection of statements that performs a specific task. It is similar to a function in other programming languages. The key difference between a function and a method is that a method is associated with an object or a class, whereas a function is not.

Default Method (void)

A default method in Java is a method that does not return any value. It is declared using the `void` keyword.


public void greet() {
    System.out.println("Hello, welcome!");
}

Method with Parameters

A method in Java can accept parameters, which are specified within parentheses after the method name and type.


public void add(int a, int b) {
    int sum = a + b;
    System.out.println("Sum: " + sum);
}

Method with Return Type

A method in Java can return a value of a specific data type using the `return` keyword.


public int multiply(int a, int b) {
    return a * b;
}

Method with Return Type and Parameters

A method can both accept parameters and return a value.


public double calculateArea(double radius) {
    return Math.PI * radius * radius;
}
